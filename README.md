## ❓ Perguntas Frequentes

**Não encontrei uma vaga agora. Posso me cadastrar?**
Sim! O banco de talentos fica disponível para futuras oportunidades alinhadas ao seu perfil.

**As vagas são para todo o Brasil?**
As oportunidades variam conforme área, projeto e localidade. Confira a Gupy para detalhes de cada posição.

**Existe possibilidade de trabalho remoto ou híbrido?**
Depende da área e do projeto. As condições são especificadas em cada vaga na Gupy.

**Como funciona o processo seletivo?**
Triagem de currículo → entrevistas com RH e lideranças técnicas → proposta. Rápido e transparente.

**Existem programas de estágio ou trainee?**
Sim! Acompanhe as vagas na Gupy para conferir oportunidades para início de carreira.
